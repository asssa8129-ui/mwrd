# مورد الطاقة المتميزة للبطاريات

موقع ثابت جاهز للرفع على أي استضافة.

## الملفات
- index.html
- assets/css/style.css
- assets/js/script.js
- assets/images/ صور SVG جاهزة للبطاريات والسيارات والماركات

## طريقة التشغيل
افتح ملف index.html مباشرة في المتصفح.

## طريقة الرفع
ارفع كل محتويات المجلد إلى public_html أو مجلد الموقع في الاستضافة.

## ملاحظات مهمة
- الصور الحالية رسومات SVG مخصصة وليست الشعارات الرسمية للماركات.
- يمكن استبدال أي صورة لاحقًا بنفس اسم الملف داخل assets/images.
- عند امتلاك دومين فعلي، يفضل إضافة رابط canonical وسايت ماب رسمي.
